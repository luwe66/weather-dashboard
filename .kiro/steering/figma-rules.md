# Figma 操作规则

## 变量绑定规则

绑定颜色变量时，以下情况**跳过不绑定**，避免视觉效果改变：
- `paint.opacity < 1`（fill 本身带透明度）
- `paint.color.a < 1`（颜色 alpha 通道不为 1）
- 渐变类型（`LINEAR`、`RADIAL` 等）

典型案例：
- 7天预报「今天」选中行：半透明背景和边框
- API 开关激活状态：半透明背景

- **严禁删除**用户 Figma 文件中的任何 frame、组件、图层或页面
- **严禁移动**用户已有的 frame 位置
- **严禁修改**用户已有 frame 的尺寸、样式或内容
- 所有操作只能是**新增**：新建 frame、新建页面、新增图层

## 导入页面时

- 每次导入生成的设计稿放在新的位置，不覆盖已有内容
- 等待页面数据完全加载后再捕获（延迟至少 8 秒）

## 读取设计稿时

- 只读取用户指定的 node-id
- 不对未被指定的节点做任何修改
