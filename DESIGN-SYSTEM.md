# 天气大屏 · 视觉设计规范

> 基于当前代码实现整理，供 Figma 设计时参考。

---

## 一、色彩系统

### 背景色

| 名称 | 色值 | 用途 |
|------|------|------|
| bg-primary | `#0a0e1a` | 页面主背景 |
| bg-secondary | `#0d1526` | 顶部栏、次级背景 |
| bg-card | `#111d35` | 卡片背景 |

> 三层背景从深到浅，营造层次感。

### 边框 & 分割线

| 名称 | 色值 | 用途 |
|------|------|------|
| border-color | `#1a3a6b` | 卡片边框、分割线、输入框边框 |

### 强调色（发光效果）

| 名称 | 色值 | 用途 |
|------|------|------|
| accent-blue | `#00d4ff` | 主强调色，城市名、图表线条、hover 状态 |
| accent-cyan | `#00ffcc` | 次强调色，数据指标数值 |
| accent-purple | `#7b61ff` | 降水概率柱状图 |

> 强调色建议配合 `box-shadow` 或 `text-shadow` 做发光效果，增强科技感。

### 文字色

| 名称 | 色值 | 用途 |
|------|------|------|
| text-primary | `#e8f4ff` | 主要文字，温度数字、城市名 |
| text-secondary | `#7a9cc0` | 次级文字，天气描述、标签 |
| text-muted | `#4a6a8a` | 弱化文字，图标、辅助说明、section 标题 |

### 状态色

| 名称 | 色值 | 用途 |
|------|------|------|
| success | `#00e676` | 正常状态、API 请求中指示灯、AQI 优 |
| warning | `#ffab40` | 警告、日出日落时间、温度条渐变终点 |
| danger | `#ff5252` | 危险、API 用量超限 |

## 二、字体

### 字体族

```
'PingFang SC', 'Microsoft YaHei', sans-serif
```

中文优先 PingFang SC（macOS），其次 Microsoft YaHei（Windows）。

### 字号规范

| 用途 | 字号 | 字重 |
|------|------|------|
| 城市名 | 28px | 700 |
| 主温度数字 | 72px | 200（超细，视觉轻盈） |
| 天气描述 | 18px | 400 |
| 数据指标数值 | 16px | 600 |
| Section 标题 | 13px | 400，全大写，letter-spacing: 1px |
| 辅助文字 | 12–13px | 400 |
| 最小文字 | 11px | 400 |

## 三、圆角

| 场景 | 圆角值 |
|------|--------|

| 进度条、滚动条 | 2–4px |
| 图标按钮 | 6px |

## 五、布局

### 整体结构

```
┌─────────────────────────────────────────┐
│              顶部栏（63px）               │
├──────────┬──────────────────┬───────────┤
│  左列     │     中列          │  右列     │
│  280px   │     flex: 1      │  280px   │
│          │                  │          │
│ 实时天气  │  24h 温度折线图   │ API 用量  │
│ 日出日落  │  7天预报          │          │
└──────────┴──────────────────┴───────────┘
```

### 顶部栏

- 高度：63px
- 背景：`bg-secondary`
- 底部边框：1px `border-color`
- 三段式布局：左（Logo + 时间）/ 中（搜索框）/ 右（状态 + 全屏）

---

## 六、图标

### 天气图标
- 来源：自有图标库（和风天气命名规范，SVG 格式）
- 路径：`/icons/{code}.svg`
- 尺寸：实时天气 32px，7天预报 24px

### UI 基础图标
- 来源：Remix Icons（line 风格）
- 路径：`/icons/ui/{name}.svg`
- 颜色：`#4a6a8a`（text-muted），hover 变 `#00d4ff`（accent-blue）
- 当前使用：`search-line`、`fullscreen-line`、`fullscreen-exit-line`

---

## 七、卡片规范

```
背景：#111d35
边框：1px solid #1a3a6b
圆角：12px
内边距：20–24px
```

Section 标题统一样式：
```
字号：13px
颜色：#4a6a8a
大写：text-transform: uppercase
间距：letter-spacing: 1px
下边距：12–16px
```

## 八、特效

### 发光效果（强调元素）
```css
/* 城市名发光 */
text-shadow: 0 0 20px rgba(0, 212, 255, 0.5);

/* 状态指示灯 */
box-shadow: 0 0 6px #00e676;
```

### 渐变（图表）
```
温度折线图填充：rgba(0, 212, 255, 0.3) → rgba(0, 212, 255, 0)
降水概率柱状图：rgba(123, 97, 255, 0.6) → rgba(123, 97, 255, 0.1)
温度区间条：#00d4ff → #ffab40
AQI 仪表盘：#00e676 / #ffeb3b / #ff9800 / #f44336 / #9c27b0
```
